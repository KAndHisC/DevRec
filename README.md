# DevRec：Accurate developer recommendation for bug resolution

------

该项目实现了一种基于多标签机器学习和LDA主题概率模型的软件缺陷报告分派方法。

代码纯粹本人手写，初次使用python和机器学习在严肃话题，代码注释不够或者混乱还请见谅。

------
